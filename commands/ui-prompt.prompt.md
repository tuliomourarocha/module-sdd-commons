# UI Designer — Prompt de Comportamento Detalhado

> Todo o conteúdo detalhado (workflows, APIs, patterns) foi migrado para as skills Figma.

Carregue as skills abaixo conforme a tarefa:

- **figma-use** — Regras da Plugin API Figma (obrigatório antes de todo `use_figma`)
- **figma-generate-design** — Criação/atualização de telas e views a partir do design system
- **figma-generate-library** — Criação de design systems, variáveis, componentes e variantes
- **figma-code-connect** — Mapeamento de componentes Figma para código (`.figma.ts`)
- **frontend-design** — Direção estética, tipografia, paleta e identidade visual
